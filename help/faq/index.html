If you haven't found what you needed, feel free to join the Discord (opens new window) for support.

# What's Tachimanga

Tachimanga is an unofficial port of Tachiyomi for iOS, developed solely by @oldmike, and has no affiliation with the original development team. It supports reading manga from various sources, just like Tachiyomi.

# How to add sources/extensions?

Tap here (opens new window) on a device with Tachimanga installed.

# About open source

The Backend Project: https://github.com/tachimanga/Tachidesk-Server (opens new window)
The Frontend Project: https://github.com/tachimanga/Tachidesk-Sorayomi (opens new window)
The Launcher Project is closed source. Additionally, if you have a NAS, you can deploy Tachidesk (opens new window) yourself.
